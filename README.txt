

1. System requirements
The programs are based in Matlab R2018a, and FreeSurfer v6.0.0, using the freesurfer (Matlab), DPABISurf(Matlab), SPM12(Matlab), and mediation(R) toolboxes.
All data were analyzed using a Linux system and an OS system.
